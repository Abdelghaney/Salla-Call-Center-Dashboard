# Salla Call Center Analytics Dashboard

## Project Overview
This project analyzes call center operations data to evaluate agent performance, service quality, call traffic, and forecast accuracy.

## Tools Used
- Python
- Pandas
- Power BI
- DAX
- Power Query

## Key KPIs
- Calls Handled Rate
- Service Level
- Abandon Rate
- ASA
- Forecast Accuracy
